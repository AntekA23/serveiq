# Sprint 9 — Pilot Onboarding Readiness

**Okres**: 25-31 maja 2026
**Cel sprintu**: Produkt mozna wdrozyc u prawdziwego klienta — onboarding, import danych, redukcja tarcia.

---

## Taski

### A1. Onboarding flow dla klubu
- Uproszczony wizard: nazwa klubu, dodanie trenerow (email invite), dodanie grup
- Trener onboarding: pierwszy login, ustawienie profilu, dodanie graczy

### A2. Import danych
- CSV import graczy (imie, nazwisko, data urodzenia, email rodzica, etap)
- Lub formularz batch-add (wielu graczy na raz)

### A3. Admin shortcuts
- Szybkie dodanie wielu graczy do grupy
- Szybkie przypisanie trenera do grupy
- Kopiowanie aktywnosci (np. powtarzajace sie zajecia)

### A4. Fix friction z demo
- Feedback z demo rozmow -> lista fix-ow
- Priorytyzacja: co blokuje uzycie vs nice-to-have

### A5. Rozmowy pilotowe
- 2-3 kandydatow na pilota zidentyfikowanych
- Draft pilot pricing wyslan
- Pilot success criteria zdefiniowane

---

## Definition of Done
- [ ] Klub moze byc setup-owany w < 30 minut
- [ ] Import graczy dziala (CSV lub batch)
- [ ] Top 5 friction points z demo naprawione
- [ ] Min. 2 kandydatow pilotowych w rozmowach
