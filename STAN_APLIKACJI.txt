1. CO TO JEST SERVEIQ?
================================================================================

ServeIQ to platforma do rozwoju mlodych tenisistow, laczaca trenerow i rodzicow.
Glowny workflow: Plan → Komunikacja → Monitoring → Ocena → Rekomendacje

Komercyjny punkt wejscia: Tennis 10 / junior club (dzieci 8-16 lat).
Architektura projektowana pod sciezke zaawansowanego zawodnika.

Model biznesowy:
  - 14-dniowy trial automatyczny
  - Free (ograniczone funkcje)
  - Premium: 39 PLN/mies
  - Family: 59 PLN/mies

3. STACK TECHNOLOGICZNY
================================================================================

BACKEND:
  Framework:      Express.js 4.21
  Runtime:        Node.js
  Baza danych:    MongoDB + Mongoose 8.9
  Autoryzacja:    JWT (access + refresh) + bcryptjs + httpOnly cookies
  Real-time:      Socket.io 4.8
  Walidacja:      Zod 3.24
  Upload plikow:  Multer
  Platnosci:      Stripe 17.4 (szkielet)
  Email:          Resend 4.0 (szkielet, loguje do konsoli w dev)
  AI:             @anthropic-ai/sdk 0.80 (Claude API)
  Bezpieczenstwo: Helmet, CORS, rate limiting
  Zmienne srod.:  dotenv

FRONTEND:
  Framework:      React 18.2 + Vite 5.4
  Routing:        React Router 6.21
  State:          Zustand 4.4
  HTTP:           Axios 1.6 (z interceptorami dla token refresh)
  Formularze:     React Hook Form 7.49 + Zod resolver
  Ikony:          Lucide React 0.303
  WebSocket:      socket.io-client 4.7
  Wykresy:        SVG (custom, bez bibliotek)

DEPLOYMENT:
  Target:         Railway
  Konfiguracja:   railway.json

JEZYK UI:         Polski (wszystkie teksty w interfejsie)
