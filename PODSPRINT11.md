# Sprint 11 — Pilot Preparation

**Okres**: 8-14 czerwca 2026
**Cel sprintu**: Pierwszy pilot moze realistycznie wystartowac — tweaki, import, feedback loop, checklist.

---

## Taski

### A1. Pilot-specific tweaks
- Dostosowanie etykiet/jezykow do kontekstu konkretnego klubu
- Konfiguracja pathway stages jesli klub uzywa wlasnych nazw
- Setup grup i trenerow

### A2. Import/setup support
- Wsparcie przy imporcie danych pilotowego klubu
- Dodanie graczy, rodzicow, trenerow
- Weryfikacja linkowania parent-child-coach

### A3. Feedback capture
- Prosty mechanizm zbierania feedbacku (formularz lub przycisk w app)
- Lub zewnetrzny: Notion/Google Form link

### A4. Finalny pass na scenariusze
- Tennis 10 scenariusz: przejsc caly flow jako klub pilotowy
- Sonia scenariusz: przejsc caly flow jako zaawansowana rodzina
- Udokumentowac luki

### A5. Minimalne analytics/logging
- Logowanie kluczowych akcji (login, tworzenie aktywnosci, dodanie notatki, publikacja przegladu)
- Prosty dashboard lub eksport do sprawdzenia uzycia
